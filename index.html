SynergyChrome

Bookmarklet that gives Synergy basic support for Chrome.

Description

The VBS version of the webapplication Synergy (v3.9, for as far as I know) only works on IE 8 natively, or IE > 8 with manually modifying the compatibility settings. This bookmarklet supplies some basic functionality in Chrome (tested) and probably Safari and Firefox too (untested).

This will only work for PSA users (limited rights users) and not for Full Suite users. I don't have access to a Full Suite user account, so I can't develop for those screens.

Just start up Synergy and run the bookmarklet.

Installation

  1. I will assume you use Chrome for all steps (tested with Chrome 36)
  2. make a bookmark of this link:
    SynergyChrome (v0.9.7)
    (Chrome: drag button to bookmarkbar)
  3. log into Synergy
  4. go to Mijn realisaties > wekelijks. This will give a loading screen that doesn't disappear.
  5. click the bookmark, changes should be visible immediately.
