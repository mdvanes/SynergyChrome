SynergyChrome

Bookmarklets that gives Synergy basic support for Chrome.

Description

The VBS version of the webapplication Synergy only works on IE 8 natively, or IE > 8 with manually modifying the compatibility settings. This bookmarklet supplies some basic functionality in Chrome (tested) and probably Safari and Firefox too (untested).

Just startup Synergy and run the bookmarklet.

Installation

  1. I will assume you use Chrome (tested with 36) for all steps
  2. creating the bookmark is not completely done, but I'm working on it. The current workaround is:
    1. create a bookmark to something (e.g. www.google.com) in Chrome
    2. right-click the bookmark and choose edit
    3. change the name to SynergyChrome
    4. remove the URL
    5. copy the contents of this file
    6. paste it as the URL
    7. save the bookmark
  3. log into Synergy
  4. go to Mijn realisaties > wekelijks. This will give a loading screen that doesn't disappear.
  5. click the bookmark, changes should be visible immediately.
